# Shivanna | Developer Portfolio

A hacker/terminal style personal portfolio website.

## 🚀 Live Demo
> Deploy to GitHub Pages (instructions below)

## 📁 File Structure
```
portfolio/
├── index.html       ← Main page
├── style.css        ← All styling
├── script.js        ← Animations & interactions
├── assets/
│   └── resume.pdf   ← Add your resume here!
└── README.md
```

## ⚙️ Setup & Deploy (GitHub Pages)

1. Create a new repo on GitHub named `portfolio`
2. Upload all these files
3. Go to **Settings → Pages**
4. Set Source to **main branch**
5. Click Save
6. Your site goes live at: `https://shivannagorkal.github.io/portfolio`

## 📬 Contact Form
Already connected to Formspree. Just works!

## 🖼️ Add Your Resume
Drop your resume PDF into the `assets/` folder and name it `resume.pdf`

## ✏️ Customization
- Colors: edit `:root` variables in `style.css`
- Add projects: copy a `.project-card` block in `index.html`
- Add certificates: copy a `.cert-card` block
